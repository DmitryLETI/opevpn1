Restart Actions

{{template "common/alert.html" .}}
Restart Server   Conditional OpenVPN Server restart will apply new configuration and re-initiate all active clients sessions.
Restart Container   Container restart of OpenVPN Server to re-apply Firewall rules and restart OpenVPN Server. Will take more time than Conditional Restart.
Restart UI   OpenVPN-UI Container Restart. Current Web session will be disconnected.

Remove Actions


DANGER ZONE! You can't rollback any actions of this section!
Remove actions can lead to descrepencies in OpenVPN-UI DB. In most cases you have to Remove DB and Restart UI after, to recreate DB.
DELETE ALL   This button will execute all the seteps below, one by one - delete PKI, Server keys, Client certificates, "*.ovpn" files and OpenVPN-UI Database.
Remove PKI   Delete PKI, all OpenVPN Server keys, and Client certificates. This will prevent clients from connecting using their current certificates!
Remove .ovpn   Remove "*.ovpn" files from the "/etc/openvpn/clients" directory.
Remove Static   Remove all static IP reservations from the "/etc/openvpn/staticclients" directory.
Remove DB   Delete the OpenVPN-UI Database. Restart OpenVPN-UI after.

Initialization Actions


DANGER ZONE! You can't rollback any actions of this section!
Be cautious, high expertise level is essential! EasyRSA Quick Guide, EasyRSA detailed Guide.
{{template "common/alert.html" .}}
INITIALIZE ALL   This button will execute all the steps below one by one - copy EasyRSA vars and create PKI, CA, Server, DH, TA and CRL.
EasyRSA VARs   Copy EasyRSA vars from "./openvpn/config/easy-rsa.vars to ./easy-rsa/pki/vars".
Initialize PKI   Initialize PKI by running "easyrsa init-pki".
Build CA   Build CA by running "easyrsa build-ca nopass".
Build Server   Generate a new Server Certificate, Key, and Encryption Files by running "easyrsa gen-req&sign-req server nopas".
Generate DH   Generate a new Diffie-Hellman key by running "easyrsa gen-dh". This step may take up to 10 minutes.
Generate TA   Generate a new openvpn TA key by running "openvpn --genkey --secret ta.key".
Generate CRL   Generate a new Certificate Revocation List (CRL) by running "easyrsa gen-crl".
