PKI Remove


DANGER ZONE! You can't rollback any actions of this section!
{{template "common/alert.html" .}}
DELETE ALL   This will delete PKI, all OpenVPN Server keys, Client certificates, "*.ovpn" files and OpenVPN-UI Database.
Remove PKI   This will delete PKI, all OpenVPN Server keys and Client certificates .
Remove .ovpn   This will remove "*.ovpn" files from "/etc/openvpn/clients" directory.
Remove Static   This will remove all static IP reservations from "/etc/openvpn/staticclients" directory.
Remove DB   This will delete OpenVPN-UI Database.
